
🎯 Python Hangman Project
This project is a console-based Hangman Game created using Python for the CodeAlpha Internship Program.

✅ Project Highlights
1. Hidden word guessing game
2. User enters one letter at a time
3. Displays correct and wrong guesses
4. Allows only limited chances
5. Simple text-based interface
  
💻 Tools & Concepts Used
1. Python Programming
2. Random Library
3. While Loop
4. If-Else Statements
5. Lists and Strings
6. User Input/Output

🚀 Steps to Execute
1. Install Python on your system
2. Open the project in VS Code
3. Open terminal
4. Execute the file using:python hangman.py

1. Improved problem-solving skills
2. Learned basic game development
3. Practiced conditional statements
4. Worked with loops and lists
